# Todo-App
Todo app based on React.js

## Setup
```bash
npm install
```

## Launch
```bash
npm start
```
Browse to http://localhost:3000/ to see the app

## Testing
To run unit tests:
```bash
npm test
```
To generate code-coverage:
```bash
npm run coverage
```
Also code-coverage report is included in the repo.
